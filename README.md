## Requisites
- Python
-> Flask

## Setup

- Clone the repository
- Install dependencies
```console
  python -m venv venv

  # for linux or mac
  source venv/bin/activate 

  # for windows
  .\venv\scripts\activate

  pip install -r requirements.txt
  ```
  - Run application
```console
    python run.py
```
